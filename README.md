# Brie Yourself
A game about the circle of life from the perspective of a block of cheese.

## Jam Details
Spring Game Jam 2022: 
- Date: 29.04.2022 - 01.05.2022
- Site: Universität Bayreuth
- Theme: "Little Boxes"
